# Breast Cancer Subtype Classification Using Gene Expression Data

This repository contains the Exploratory Data Analysis (EDA) and baseline modeling for Checkpoint 1 of the CS228 project.

## How to Run
1. Open the `CS228_Checkpoint1_EDA.ipynb` file.
2. Click the **"Open in Colab"** button or upload the file directly to [Google Colab](https://colab.research.google.com/).
3. Ensure the dataset is loaded and run all cells to view the biometric metrics (FAR, FRR, and Accuracy).

## Project Overview
- **Dataset:** 151 samples with 54,677 genomic features.
- **Model:** Balanced Random Forest Classifier.
- **Goal:** Classify breast cancer subtypes with high biometric security standards.
